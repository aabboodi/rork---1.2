# Security Components

مجموعة شاملة من المكونات الأمنية المصممة لتوضيح وشرح الميزات الأمنية للمستخدمين العاديين.

## المكونات المتاحة

### 1. SecurityDashboard
لوحة تحكم رئيسية تعرض حالة الأمان العامة للتطبيق.

**الميزات:**
- عرض حالة النظام الأمني
- مقاييس الأمان الأساسية
- الميزات الأمنية المفعلة
- نصائح أمنية سريعة
- إجراءات سريعة

### 2. SecurityExplainer
مكون تفاعلي لشرح الميزات الأمنية بطريقة مبسطة.

**الميزات:**
- شرح مفصل لكل ميزة أمنية
- أمثلة عملية
- فوائد واضحة للمستخدم
- نصائح للاستخدام الأمثل

### 3. SecurityOnboarding
دليل تفاعلي للمستخدمين الجدد لفهم الميزات الأمنية.

**الميزات:**
- جولة تفاعلية خطوة بخطوة
- شرح مبسط للمفاهيم الأمنية
- إمكانية التنقل بين الخطوات
- تصميم جذاب ومفهوم

### 4. SecurityHealthWidget
عرض مرئي لحالة الأمان العامة مع مقاييس تفصيلية.

**الميزات:**
- درجة أمان شاملة
- مقاييس فردية لكل ميزة
- رسوم بيانية تفاعلية
- مؤشرات الاتجاه

### 5. SecurityTipsCarousel
عرض دوار للنصائح الأمنية اليومية.

**الميزات:**
- نصائح أمنية متنوعة
- تمرير تلقائي
- تصنيف حسب الأهمية
- إجراءات قابلة للتنفيذ

### 6. SecurityNotificationBanner
نظام إشعارات أمنية ذكي.

**الميزات:**
- إشعارات متنوعة الأنواع
- إخفاء تلقائي
- إجراءات سريعة
- تصميم جذاب

### 7. SecurityStatusIndicator
مؤشر حالة أمنية مع شرح تفاعلي.

**الميزات:**
- حالات متعددة (آمن، تحذير، حرج)
- شرح تفاعلي عند النقر
- أحجام متعددة
- ألوان واضحة

### 8. SecurityTooltip
نصائح سريعة وشروحات للميزات الأمنية.

**الميزات:**
- شرح سريع عند النقر
- أنواع متعددة (معلومة، تحذير، نجاح)
- تصميم أنيق
- سهولة الاستخدام

## الاستخدام

```tsx
import {
  SecurityDashboard,
  SecurityExplainer,
  SecurityOnboarding,
  useSecurityNotifications
} from '@/components/security';

function MySecurityScreen() {
  const { showSecuritySuccess } = useSecurityNotifications();
  
  return (
    <View>
      <SecurityDashboard />
      <SecurityExplainer visible={true} onClose={() => {}} />
    </View>
  );
}
```

## التخصيص

جميع المكونات قابلة للتخصيص من خلال:
- الألوان والثيمات
- النصوص والترجمات
- السلوكيات والتفاعلات
- البيانات المعروضة

## إمكانية الوصول

تم تصميم جميع المكونات مع مراعاة:
- دعم قارئات الشاشة
- تباين الألوان المناسب
- أحجام النصوص القابلة للقراءة
- التنقل بالكيبورد

## الأمان

المكونات مصممة لـ:
- عدم تسريب معلومات حساسة
- عرض معلومات آمنة فقط
- توجيه المستخدمين لأفضل الممارسات
- تعزيز الوعي الأمني
